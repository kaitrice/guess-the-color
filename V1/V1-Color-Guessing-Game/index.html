What are color codes?

RGB color model represents red, green, blue primary colors of light added together in various ways to reproduce every color. Each value ranges from 0 to 255.

Represented as rgb(red, green, blue)
Example: rgb(0, 0, 0) would represent black
Example: rgb(255, 255, 255) would represent white

HEX color model is another notation where each of the three decimal values are converted into a two-digit hexadecimal code, resulting in a 6-digit hexadecimal code.

Example: #000000 would represent black
Example: #ffffff would represent white

How to play:

Based on the color value above, select the color you think is being represented. Depending on what level you choose, there are either 3, 6, or 9 points availble each round. Each incorrect answer will decresase the availble points by 1 and the square will disappear.
